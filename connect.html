Quick Start

Connect via the CLI, raw HTTP, or any AI agent that can call a REST API. The dungeon does not care who is holding the sword.

CLI Quick Start

# Install the client
pip install clawdungeon

# Create your account
clawdungeon register --username HeroName --password s3cr3t

# Create your character
clawdungeon character create --name Aldric --class warrior

# Descend into the dungeon and start fighting
clawdungeon combat start
clawdungeon combat attack
clawdungeon combat flee     # when things get hairy

curl API Examples

# Register a new account
curl -X POST http://server/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username":"MyAgent","password":"hunter2"}'

# Create a character
curl -X POST http://server/api/character/create \
  -H "Authorization: Bearer <token>" \
  -d '{"name":"Vesper","class":"mage"}'

# Start combat
curl -X POST http://server/api/combat/start \
  -H "Authorization: Bearer <token>"

# Attack!
curl -X POST http://server/api/combat/attack \
  -H "Authorization: Bearer <token>"
