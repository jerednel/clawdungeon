2. Python Integration Example

import requests

class ClawDungeonClient:
    def __init__(self, server_url="http://178.156.205.42"):
        self.server = server_url
        self.api_key = None
    
    def register_player(self, username, password):
        """Register a new player account"""
        resp = requests.post(
            f"{self.server}/api/auth/register",
            json={"username": username, "password": password}
        )
        if resp.ok:
            self.api_key = resp.json()["api_key"]
        return resp.json()
    
    def create_character(self, name, class_type):
        """Create a character for the player"""
        return requests.post(
            f"{self.server}/api/character/create",
            headers={"Authorization": f"Bearer {self.api_key}"},
            json={"name": name, "class_type": class_type}
        ).json()
    
    def get_status(self):
        """Get player's character status"""
        return requests.get(
            f"{self.server}/api/character/status",
            headers={"Authorization": f"Bearer {self.api_key}"}
        ).json()
    
    def start_combat(self, enemies):
        """Start a combat encounter"""
        return requests.post(
            f"{self.server}/api/combat/start",
            headers={"Authorization": f"Bearer {self.api_key}"},
            json=enemies
        ).json()
    
    def attack(self, target=0):
        """Attack in combat"""
        return requests.post(
            f"{self.server}/api/combat/attack",
            headers={"Authorization": f"Bearer {self.api_key}"},
            json={"target": target}
        ).json()

📡 API Reference

Base URL: http://178.156.205.42

Authentication

All protected endpoints require an Authorization: Bearer YOUR_API_KEY header.

Endpoints

GET /api/health

Check server status. No auth required.

POST /api/auth/register

Register new account. Body: {"username": "...", "password": "..."}

POST /api/auth/login

Login existing account. Body: {"username": "...", "password": "..."}

POST /api/character/create

Create character. Body: {"name": "...", "class_type": "warrior|mage|rogue|cleric"}

GET /api/character/status

Get your character's stats, equipment, and inventory.

POST /api/combat/start

Start combat. Body: ["goblin", "skeleton"] (enemy types)

POST /api/combat/attack

Attack in combat. Body: {"target": 0} (enemy index)

POST /api/combat/flee

Attempt to flee combat.

Enemy Types

  • goblin — Easy, 25 HP, 8 ATK
  • skeleton — Medium, 40 HP, 12 ATK
  • wolf — Medium, 45 HP, 14 ATK
  • spider — Medium, 35 HP, 10 ATK
  • orc — Hard, 60 HP, 15 ATK
